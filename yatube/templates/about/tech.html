Инструменты и стек.


  • Python
  • Django
  • HTML
  • CSS
  • Bootstrap
  • Unittest

  • Python - мой основной язык программирования. Я использую его для написания бэкенд-кода, а также для автоматизации различных задач.
  • Django - это веб-фреймворк на языке Python, который я использую для создания веб-приложений. Он предоставляет множество полезных инструментов для ускорения разработки и облегчения ее процесса.
  • HTML и CSS - это основы веб-разработки. HTML используется для создания структуры веб-страницы, а CSS - для ее оформления и стилизации.
  • Bootstrap - это фреймворк для CSS, который предоставляет готовые стили и компоненты для ускорения разработки веб-сайтов. Он помогает сделать веб-сайт адаптивным и приятным для пользователей.
  • Unittest - это библиотека для тестирования кода на языке Python. Я использую ее для написания автоматических тестов, которые помогают мне быстро находить и исправлять ошибки в коде.

Конечно, это не все инструменты, которые я использую в своей работе, но они являются основными. Я всегда открыт для новых технологий и готов учиться новому, чтобы стать еще более квалифицированным разработчиком.
